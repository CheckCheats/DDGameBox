🔐 如何获取 GitHub Token

  1. 打开 github.com/settings/tokens
  2. 点击 Generate new token → Generate new token (classic)
  3. Note 填任意名称，Expiration 选 No expiration
  4. 勾选权限: ☑ repo (全部) + ☑ workflow
  5. 点击底部 Generate token
  6. 复制生成的 ghp_xxxx... 粘贴到上方输入框

⚠️ Token 仅保存在浏览器本地，不会上传到任何服务器。用于调用 GitHub API 触发后台下载。
